Welcome to D Programming Tutorial

D is a systems programming language with C-like syntax and static typing. Let's explore the basics!

Getting Started with D

To start programming in D, you'll need a D compiler installed on your system. DMD (Digital Mars D) is the most commonly used compiler for D programming.

Your First D Program

Let's write a simple "Hello, World!" program:


import std.stdio;

void main()
{
writeln("Hello, World!");
}

Save this code in a file with a .d extension, such as hello.d. Then, compile and execute it:


dmd hello.d -of=hello
./hello

You should see the output:


Hello, World!

Basic Syntax

D syntax is similar to other C-style languages. It includes statements, variables, data types, functions, and more. Here are some essential syntax rules:

Functions and Procedures

Functions in D are defined using the `void` keyword for procedures and specify the return type for functions. They can have parameters and a function body.

Modules and Imports

D programs are organized into modules, which are files containing code. You can import modules using the `import` keyword to use their functionality in your program.

Arrays and Slices

D provides arrays and slices for working with collections of elements. Arrays have a fixed size, while slices are dynamically sized views into arrays or other data structures.

Object-Oriented Programming (OOP)

D supports object-oriented programming (OOP) concepts like classes, interfaces, inheritance, and polymorphism, allowing you to write modular and reusable code.

Further Learning

D is a versatile language suitable for various types of projects, including system programming, web development, and scientific computing. Explore online resources, tutorials, and books to master D programming.
