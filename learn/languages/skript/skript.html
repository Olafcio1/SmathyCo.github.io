Skript

Welcome to Skript Tutorial

Skript is a powerful scripting language used to create plugins for Minecraft servers running Bukkit/Spigot. Let's get started!

Introduction to Skript

Skript is a scripting language that allows server administrators to customize gameplay on Minecraft servers without the need for extensive plugin development. It simplifies the process of creating custom features and functionality.

Setting Up Skript

To use Skript, you need to have a Minecraft server running Bukkit or Spigot. Once your server is set up, you can install the Skript plugin, which allows you to write and execute Skript scripts directly in your server.

Hello, World! in Skript

Let's start with a simple "Hello, World!" script in Skript:


on join:
    send "Hello, World!" to player

This script sends "Hello, World!" to a player whenever they join the server.

Basic Syntax

Skript syntax is designed to be human-readable and intuitive. It uses a natural language-like syntax with keywords and expressions that resemble English sentences.

Events and Conditions

In Skript, events are triggers that cause scripts to run, and conditions are used to check if certain criteria are met before executing actions. Here's an example:


on player death:
    if victim is a player:
        broadcast "%victim% died!"

This script broadcasts a message when a player dies.

Variables and Data Types

Skript supports various data types including strings, numbers, booleans, and lists. Variables can be used to store and manipulate data within scripts.


on join:
    # The player datatype example
    set {_plr} to player

    # The string datatype example
    set {_nick} to player's name
    set {_str} to "&6I like Skript!"

    # The boolean datatype example
    if player is online:
        set {_isOnline} to true
    else:
        send "&6Hey, this isn't possible! You just joined, and are offline???"

    # The list datatype example
    set {_players::*} to online players
    send "&6Currently online players: %{_players::*}%"

Loops and Iteration

Skript provides loop structures like foreach loops and while loops for iterating over collections of data or repeating actions until a condition is met.


command /kickall [<text>]:
    trigger:
        send {_players::*} to online players
        loop {_players::*}:
            if loop-player has permission "kickall.bypass":
                continue
            else:
                kick loop-player due to colored arg-1
        send "&6Kicked all players."

Functions and Effects

Functions and effects in Skript allow you to define custom actions and behaviors. Functions can take parameters and return values, while effects perform actions in the game world.


function tpToSpawn(plr: player):
    teleport {_plr} to {spawn}
    send "&6Teleported."

command /spawn:
    executable by: players
    trigger:
        tpToSpawn(player)

on respawn:
    tpToSpawn(victim)

Expressions and Operators

Skript supports various expressions and operators for performing calculations, comparisons, and logical operations. These allow you to manipulate data and control the flow of your scripts.

Error Handling

Skript provides error handling mechanisms to handle unexpected situations and errors gracefully. You can use try-catch blocks to catch and handle exceptions in your scripts.

Further Learning

Skript offers a wide range of features and possibilities for customizing Minecraft servers. There are many advanced topics like event scripting, GUI creation, database integration, and plugin development to explore.

Happy scripting with Skript!
