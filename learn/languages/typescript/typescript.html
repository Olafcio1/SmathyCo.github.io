TypeScript

Welcome to TypeScript Tutorial

TypeScript is a superset of JavaScript that adds static types to the language. Let's dive in!

Introduction to TypeScript

TypeScript is a statically typed superset of JavaScript that compiles to plain JavaScript. It adds optional static types, interfaces, enums, and other features to JavaScript, making it more robust and scalable for large-scale applications.

Installing TypeScript

To start using TypeScript, you need to install Node.js and npm (Node Package Manager) if you haven't already. Then, you can install TypeScript globally using npm:


npm install -g typescript

Hello, World! in TypeScript

Let's start with a simple "Hello, World!" program in TypeScript:


console.log("Hello, World!");

This code will log "Hello, World!" to the console when executed.

Basic Syntax

TypeScript syntax is similar to JavaScript with additional static typing features. It includes primitives like number, string, boolean, and complex types like arrays, tuples, enums, and objects.

Variables and Constants

In TypeScript, you can declare variables using the var, let, or const keywords. Here's an example:


let myVariable: number = 42;
const myConstant: string = "Hello";

This declares a variable named "myVariable" with a type of number and assigns it the value 42. It also declares a constant named "myConstant" with a type of string and assigns it the value "Hello".

Data Types

TypeScript supports various data types including number, string, boolean, arrays, tuples, enums, any, void, and more. You can also create custom data types using interfaces and classes.

Control Flow

TypeScript supports control flow structures like if statements, for loops, while loops, switch statements, and more. These allow you to control the flow of your program based on conditions and iterate over collections.

Functions

Functions in TypeScript are similar to JavaScript functions but can also specify parameter types and return types. Here's an example:


function greet(person: string): string {
return "Hello, " + person;
}

This function takes a string parameter named "person" and returns a string greeting.

Classes and Interfaces

TypeScript supports object-oriented programming concepts like classes, interfaces, inheritance, access modifiers, and more. Classes can implement interfaces and extend other classes.

Error Handling

TypeScript provides error handling mechanisms like try-catch blocks for handling exceptions and errors. It also supports throwing and catching custom errors using classes or interfaces.

Further Learning

TypeScript is a powerful language that can greatly enhance your JavaScript development experience. There are many advanced topics like generics, decorators, modules, and asynchronous programming to explore.

Happy coding with TypeScript!
